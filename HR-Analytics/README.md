# HR Analytics Dashboard | Power BI

This Power BI project analyzes HR data to understand employee attrition, job satisfaction, and workforce distribution.

## Key Insights
- Overall employees, active employees & attrition rate
- Department-wise attrition
- Attrition by age group and gender
- Job satisfaction rating by job role
- Education field-wise attrition

## Tools & Skills
- Power BI
- DAX
- Data Modeling
- Data Visualization
